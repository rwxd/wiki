# JSON

## Usage

### Strings

```json
{
	"my_key": "my_string" 
}
```

### Integer and floats

```json
{
	"my_int": 2,
	"my_float": 3.5
}
```

### Array

```json
{
	"my_list": ["test", 5, "test3"]
}
```

### Objects
```json
{
	"my_object": {
		"name": "Test Object",
		"childs": [
			{
				"name": "Child object 1"
			}
		]
	}
}
```

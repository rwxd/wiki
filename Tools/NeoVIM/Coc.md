# Coc in Neovim

## Coc Plugins
Use in init.vim
```bash
let g:coc_global_extensions = [
            \ 'coc-pyright',
            \ 'coc-prettier',
            \ 'coc-git',
            \ 'coc-json',
            \ 'coc-docker',
            \ 'coc-yaml',
            \ 'coc-html',
            \ 'coc-sh',
            \ 'coc-go',
            \ '@yaegassy/coc-ansible',
            \ ]
```

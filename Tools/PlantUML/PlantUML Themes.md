# PlantUML Themes

An overview of all available themes can be seen in the [Theme Gallery](https://bschwarz.github.io/puml-themes/gallery.html).

## Usage
Set a theme
```plantuml
@startuml
!theme spacelab
a -> b
b -> c
@enduml
```

```plantuml
@startuml
!theme spacelab
a -> b
b -> c
@enduml
```

Get all themes with `help themes`

```plantuml
help themes
```
